클라우드 컴퓨팅 서비스 유형
- 클라우드 공급자가 관리해주는 범위에 따라 IaaS, Paas, SaaS가 있음
- IaaS : Infrastructure as a Service
- PaaS : platform as a Service
- SaaS : Software as a Service

Public Cloud
- 클라우드 제공자가 구축, 소유, 관리함.
- 불특정 다수가 사용함.
- 공유 자원을 가상화를 통해 분리함.
- 비용이 저렴하며, 확장이 용이함.

Private Cloud
- 기업 또는 조직이 구축, 소유, 관리함.
- 특정 사용자만 독립적으로 사용함.
- 자원이 외부와 공유되지 않음.
- 비용이 높으며, 확장이 제한적임.

아마존 웹 서비스(AWS)
- Amazon의 자회사
- 200개 이상의 다양한 퍼블릭 클라우드 컴퓨팅 서비스를 제공함.
- 서비스 라인업 : 컴퓨팅, 스토리지, 데이터베이스, 네트워킹 및 콘텐츠 전송, 보안 및 자격 증명 등

AWS 글로벌 클라우드 인프라
- 2024년 10월 기준, 전세계 34개의 리전과 108개의 가용 영역을 통해 글로벌 클라우드 인프라를 제공함.

리전과 가용 영역
- 가용 영역(Availability Zone)
- 물리적으로 분리된 데이터 센터의 집합
- 리전(Region)
- 지리적으로 분리된 가용 영역의 집합
- 리전에 가용영역이 포함됨
